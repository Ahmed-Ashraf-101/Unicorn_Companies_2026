# Unicorn_Companies_2026

Web scraping [The latest updated list of unicorn companies across the world](https://www.cbinsights.com/research-unicorn-companies) using BeatifulSoup library.

Then conducting robust Exploratory Data Analysis (EDA) as follows:

1- Adjusting the formatting for the features (datetimee representations, objects, integers or floating values).
2- Handling the missing data points and duplicated values.
3- Bucketizing or binning the required data fields to extract new data fields for the size of the company (Unicorn - Decacorn - Hectocorn).
4- Dealing with the outliers and identifing whether they are Global, Contextual or Collective.
5- Extracting all the important trends and insights from the dataset.
6- Vizualizing the insights using Plotly library (6 graphs have been exported for your review).

Finally, crafting well-rounded summarization for the whole analysis into PowerBI Dashboard.
